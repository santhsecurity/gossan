# gossan-subdomain

> Part of the [Santh](https://santh.dev) security research ecosystem.

Subdomain discovery scanner for gossan (CT logs, Wayback, permutations, DNS bruteforce) — part of the Santh security research ecosystem.

Part of [gossan](https://github.com/santhsecurity/gossan).
